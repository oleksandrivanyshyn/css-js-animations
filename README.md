# CSS & JS Animation Gallery 🎨

A massive collection of over 100+ custom UI effects, animations, and components built using pure CSS and Vanilla JavaScript.

## 🚀 Key Features
* **100+ Animations**: Includes creative loaders, buttons, cards, and text effects.
* **Automated Build**: Features a `build.js` script that scans directories and updates the gallery.
* **Pure Tech**: Zero dependencies—built with standard CSS3 and Vanilla JS.
* **Deployment Ready**: Configured for instant hosting on platforms like Vercel.

## 🛠️ Tech Stack
* **Styling**: CSS3 (Clip-path, Keyframes, Transforms, Transitions)
* **Logic**: Vanilla JavaScript
* **Build System**: Node.js (File System module)

## 🏁 Getting Started

### 1. Installation
```bash
git clone https://github.com/oleksandrivanyshyn/css-js-animations.git
cd css-js-animations
npm install
```

### 2. Build the Gallery
Whenever you add new animations, run the build script to update the index:
```bash
npm run build
```

### 3. Local Preview
Simply open the generated `index.html` file in any modern browser.

## 📁 Repository Structure
* `/animations`: Creative loaders and environmental effects.
* `/buttons`: Interactive hover and click states.
* `/cards`: Product and content display layouts.
* `/clip`: Advanced effects using the CSS `clip-path` property.
* `/images`: Dynamic image hover and transition effects.
* `/javascript`: Logic-driven UI components and scroll animations.
* `/menu`: Cool menu effects.
* `/other cool effects`: Smoky text, glitch effects, and experimental UI.
* `build.js`: The engine that generates the gallery UI.
